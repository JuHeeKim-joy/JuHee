## 📌 스터디 규칙
- 주 1회 대면, 대면 날짜 전 마감기한 잡고 2문제 풀기.
- 대면 스터디 때 1문제 풀기 및 총 3문제 코드 리뷰

<br/>

## 🔍 참여 방법
1. 이 저장소를 `fork` 한다.
2. 생성된 저장소에 이름, 이름 하위 폴더로 WeeK주차 로 폴더를 생성한다. ex) EuiJoong\Week01 <- 1주차 폴더
3. 알고리즘 풀이 후 원본 저장소에 `PR`를 보낸다.

<br/>

## 📁 Repository 폴더 구조
```
이름/주차/코드.java
```

- 💡 예시: `EuiJoong/Week01/덧셈과 뺄셈.java`
  
<br/>

## ⚠️ commit 컨벤션

```
git commit -m "Upload: 이름 / 날짜 / [플랫폼_문제제목_문제번호] / 시간"
```

- 💡 예시: `"Upload : 김싸피 / 240725 / [BOJ_몰라_1234] / 128ms"`
